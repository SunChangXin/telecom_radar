# 通信前沿情报

该上下文描述如何组织通信技术动态，使内容出处与研究主题保持清晰分离。

## Language

**来源**:
内容的发布或收录出处，例如 arXiv、3GPP、ITU、IEEE ComSoc、Ericsson 或 GitHub。
_Avoid_: 会议、领域、分类

**领域**:
内容涉及的通信技术主题。一条内容可以关联多个领域。
_Avoid_: 来源、会议

**主领域**:
与内容标题、标签和摘要匹配度最高的领域，用于唯一归档和数量统计。
_Avoid_: 唯一领域

**关联领域**:
除主领域外，同一内容同时涉及的其他技术主题。
_Avoid_: 次要标签

**未分类**:
当前分类词表无法可靠判断其技术主题的内容，需要补充规则或人工审阅。
_Avoid_: 其他

**内容类型**:
内容在情报工作流中的载体类型，分为论文、标准组织、产业资讯和开源项目。
_Avoid_: 领域、来源

**中文导读**:
网页中用于快速判断内容价值的中文说明。没有模型能力时仅概括已识别领域和技术要素，不冒充原文翻译。
_Avoid_: 原文摘要、自动翻译
